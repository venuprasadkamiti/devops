RUPA TURBO RUN

An upgraded single-file 3D highway racer with braking, engine audio, checkpoint racing, changing worlds, and local-versus mode.

Single Player

Drive forever through an endless procedural highway. Reach checkpoints to unlock the next graphic world and survive for the highest score.

Local Versus

Two players share one screen and race for checkpoint wins. First to 3 wins takes the crown.

Controls

P1: A / D steer, S brake, Space nitro. P pause, R restart.

Versus Keys

P2: J / L steer, K brake, I nitro. Mobile controls operate only Player 1.

This stays a single deployable HTML file and uses WebAudio for engine, nitro, brake, crash, and checkpoint sounds. Three.js loads from CDN.
